# Banking Credit Risk Power BI Dashboard

## Overview
A data analytics dashboard providing insights into loan origination, approvals, and risk analysis for a banking institution.

## Key KPIs
- NPA Ratio
- Loan Approval Rate
- Customer Risk Segmentation
